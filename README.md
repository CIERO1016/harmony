# Harmonies ソロ（自作版）

ボードゲーム「Harmonies」の 1 人用（ソロ）を、フレームワーク不使用の静的 Web アプリとして実装したものです。
公式のイラスト・ロゴは使用せず、ゲームの仕組みのみをオリジナルデザインで再現しています。

## 遊び方

- ブラウザで `index.html` を開くだけで動作します（PC 推奨）。
- 進行状況は自動でブラウザに保存されます（localStorage）。

### ルール概要

- 中央ボードのスペースから 3 トークンを取得し、個人ボード（23 マス）に配置。
- トークンは 6 種（川・山・木・葉・畑・建物）で、種類ごとに積み重ねルールがある。
- 動物カード（20 種）：条件パターンが盤上にできたらキューブを配置して得点。
- 精霊カード：開始時に 1 枚選択（または「精霊なし」）。
- 地形得点：山/葉＝高さ、畑＝連結グループ、建物＝隣接色数、川＝最長経路長。
- ソロ勝利点：ゲーム得点（40→1 … 160→8）＋盤面（A 面 +1）＋精霊タイプ。
